## 📈 Data Quality Standards

### Field Requirements by Type

**Fleet Fields (8 required/optional):**

```
✓ Required:  name, slug, description, capacity, image, amenities
✓ Optional: services, seo
✓ Localized: name, slug, description, amenities
```

**Service Fields (7 required/optional):**

```
✓ Required:  title, slug, description, hero_image, service_points
✓ Optional: fleets, seo
✓ Localized: title, slug, description, service_points
```

**Article Fields (8 required/optional):**

```
✓ Required:  title, slug, description, content, image, publishedAt
✓ Optional: categories, dynamic_zone, seo
✓ Localized: title, slug, description, content
```

---

### SEO Standards

**Meta Title:**

- Length: 50-60 characters
- Include main keyword
- Natural language
- Descriptive

**Meta Description:**

- Length: 155-160 characters
- Include keywords
- Call-to-action optional
- Unique per page

**Images:**

- OG Image: 1200x630px
- File size: < 200KB
- Format: JPG/PNG
- Alt text: Always required

---

### Naming Conventions

**Slugs (URLs):**

- Format: `kebab-case` (lowercase-with-hyphens)
- Example: `airport-transfer-service`
- Max length: 50 characters
- Must be unique per content type

**Image Filenames:**

- Format: `YYYY-MM-content-type.jpg`
- Example: `2025-01-airport-transfer.jpg`
- Descriptive and searchable

**Categories:**

- Format: `Title Case`
- Example: `Travel Tips`, `How-To Guides`
- Consistent terminology
